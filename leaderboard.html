Ranking System

1. Point are calculated by running the python script to fetch the total count of pull request/issues created on the repositories during the time intervl of event and this leaderboard is updated manually.
2. If you are a TechnoJam Core Member your name will not appear on the leaderboard.
3. Even if you have not registered your name might appear on the leadeboard but you will not be eligible for prizes.
4. Point System
         a. Pull Request   5
         b. Issues          2
5. Merged/Open/Not Merged all carry equal points but will be used to breakup the tie.
         i.e Merged > Open > Not Merged
